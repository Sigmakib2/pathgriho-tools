What is Country Code?

Country codes, mainly country calling codes or dial-in codes are some number that are being used before a mobile number to reach the destination’s mobile in case of international codes. It is numeric geographical or in short geocodes. The structure of this a mobile number is “Prefix number, generally it is 00 or (+) sign + the international calling code + the user number that are only used to make a local call (This is not for Landline, for landlines you need to add a area code after the country code).

For example, if you want to make a call to Bangladesh from Saudi Arabia, you will dial the number as 0088016292*****. Here 880 is the international calling code of Bangladesh.

What is ISO Code?

ISO code is being used to represent the names of countries. Standard ISO code do not contain fixed number of characters. It has characters that are needed to represent the country’s name properly? ISO 3166-1 Standard has 249 ISO codes from where 193 are member of UN. It can be of Alpha-2 code or Alpha-3 code or even numeric. For example, Bangladesh’s Standard ISO code is BD whereas 3 Character Code is BGD and number code is 050.

When We Need Them?

International Calling Code: We need international calling codes to make an international call. In case of local calls, we do not need Country code, but for international calls via mobile phone or land phone, they are mandatory.

ISO Code: We need iso code for form fill up, registration, and many other things. Actually, it is different for each country. So, it helps to identify the country of a candidate.
